# Плагин инициализации проекта

Плагин создает базовую структуру нового проекта, включая:

- Структуру директорий
- Базовые конфигурационные файлы
- Примеры сервисов
- Go модуль

## Опции

- project (string, обязательная) - имя проекта
- module (string, опциональная) - имя Go модуля (по умолчанию: имя проекта)
- service (string, опциональная) - имя сервиса (по умолчанию: "someService")
- dir (string, опциональная) - директория для создания проекта (по умолчанию: ./<project>)
